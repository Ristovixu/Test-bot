# Telegram Bot для записи на маникюр

🤖 Телеграм бот для мастера по маникюру с системой записи, напоминаниями и админ-панелью.

## ✨ Возможности

### Для клиентов:
- 📅 Запись на маникюр с выбором даты и времени
- ✅ Проверка подписки на канал (обязательная)
- 💰 Просмотр прайс-листа
- 📸 Просмотр портфолио
- ❌ Отмена записи
- 🔔 Автоматические напоминания за 24 часа

### Для администратора:
- ➕ Добавление рабочих дней и слотов
- ➖ Удаление слотов
- 🔒 Закрытие дней
- 📋 Просмотр расписания
- ❌ Отмена записей клиентов
- 📢 Автоматические уведомления о новых записях

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации
```bash
cp config.example.py config.py
```

Отредактируйте `config.py` и заполните:
- `BOT_TOKEN` - токен от [@BotFather](https://t.me/BotFather)
- `ADMIN_ID` - ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot))
- `NOTIFICATION_CHANNEL_ID` - ID канала для уведомлений
- `CHANNEL_ID` - username канала для обязательной подписки

### 4. Запуск бота
```bash
python bot.py
```

## 📁 Структура проекта

```
├── bot.py                 # Основной файл бота
├── bot_simple.py          # Упрощенная версия с большими таймаутами
├── config.example.py      # Пример конфигурации
├── requirements.txt       # Зависимости Python
├── database/             # Работа с базой данных
│   ├── __init__.py
│   └── db.py
├── handlers/             # Обработчики команд
│   ├── __init__.py
│   ├── admin.py          # Админ-панель
│   ├── user.py           # Пользовательские команды
│   └── subscription.py   # Проверка подписки
├── keyboards/            # Клавиатуры
│   ├── __init__.py
│   └── inline.py
└── utils/                # Утилиты
    ├── __init__.py
    ├── scheduler.py      # Планировщик напоминаний
    └── states.py         # FSM состояния
```

## 🛠️ Используемые технологии

- **aiogram 3** - фреймворк для Telegram ботов
- **aiosqlite** - асинхронная работа с SQLite
- **APScheduler** - планировщик задач
- **SQLite** - база данных
